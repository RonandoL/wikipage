Branching

Conditional statements are used to perform different actions based on different conditions.

Syntax of Branching

  $(document).ready(function() {
    var age = parseInt(prompt("How old are you?"));

    if (age >= 21) {
      $('#drinks').show();
    } else {
      $('#under-21').show();
    }
  });
