Forms

HTML forms are used to collect user input. Form elements are different types of input elements, checkboxes, radio buttons, submit buttons, and more.

Syntax

        
HTML Syntax for .submit() form

        <form id="target" action="destination.html">
          <input type="text" value="Hello there">
          <input type="submit" value="Go">
        </form>
        <div id="other">
          Trigger the handler
        </div>

        
jQuery Syntax

        $( "#target" ).submit(function( event ) {
          alert( "Handler for .submit() called." );
          event.preventDefault();
          });
